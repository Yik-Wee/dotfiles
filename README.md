## Main stuff
Network: networkmanager, network-manager-applet

Bar: polybar
File Manager: nemo, nemo-folder-icons
Compositor: picom
App Launcher: rofi
Notification Manager: dunst
Screnshotting: gscrenshot
Image Viewer: gthumb
Text editors: nano, vim
